# network-security-protocol
How to run client server program:

1. Run cmd 'python3 server.py' to start server

2. Run cmd 'python3 client.py' to start client

3. Select option 2 to generate public and private key for username

4. Select option 1 to connect to server and type username

5. Type messages

6. To switch user type 'change()' 

7. To exit type 'quit()'
